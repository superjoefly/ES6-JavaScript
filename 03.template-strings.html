Template Strings

Template Literals

Template literals are string literals that allow embedded expressions. We can use multi-line strings and string-interpolation featrures with them. Template literals are enclosed by back-ticks (` `) instead of double quotes. Template literals can contain placeholders which are indicated by the dollar-sign and curly braces ( ${expression} ):


      function dateMessage() {
        const year = new Date().getFullYear();

        // Using the variable:
        return `The year is ${year}`;

        // Using the expression itself:
        // return `The year is ${new Date().getFullYear()}`;
      }

      console.log(dateMessage());

Readability

Using template literals can make code more readable:


      const device_id = 3;
      const guid = 20;
      const username = 'James';

      const data = '{"device_id: " + device_id + "guid " + guid + "username " + username}';

      console.log(data);

      // Refactored with template literals:
      const myData = `{
        "device_id " ${device_id},
        "guid" ${guid},
        "username " ${username}
      }`;

      console.log(myData);

Note: variables by themselves don't need a template string:


      const year = 2017;

      let yearMessage = `The year is ${year}`;
      console.log(yearMessage); // => The year is 2017

      yearMessage = year; // variable by itself
      console.log(yearMessage); // 2017

Refactoring

We can refactor the following function using template literals:


      // Non-refactored
      function doubleNumber(number) {
        return 'Your number doubled is ' + (2 * number);
      }

      // Refactored using template literals:
      function doubleNumber(number) {
        return `Your number doubled is ${2 * number}`;
      }

      console.log(doubleNumber(2)); // => Your number doubled is 4

Another Refactoring Example:


      function fullName(firstName, lastName) {
        console.log(firstName + ' ' + lastName);
      }

      // Refactored using template literals:
      function fullName(firstName, lastName) {
        console.log(`${firstName + ' ' + lastName}`);
      }

      fullName('John', 'Doe'); // => John Doe
